# 🐶 Steph's Dog Gallery

## Overview

Steph's Dog Gallery is a simple React application that uses a public API to display random dog images. Users can click a button to load a new dog image each time. This project demonstrates API integration, React state management, and responsive design.

I also added a personal touch inspired by my dog Oreo, who is a Shih Tzu.

---

## Features

- Fetches random dog images from a public API
- Button to load new images
- Loading state for better user experience
- Responsive design for mobile and desktop
- Clean layout using CSS Grid
- Footer with GitHub repository link

---

## API Used

https://random.dog/woof.json

---

## Technologies Used

- React (JavaScript)
- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API

---

## How to Run

1. Open the project in CodeSandbox
2. Click "Run"
3. Click the button to load new dog images
